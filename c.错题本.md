## 默写 Vue 生命周期
  ```js
  vue 生命周期
  vue2:
  beforeCreate
  created
  beforeMount
  beforeMounted            ---------> mounted
  beforeUpdate
  updated
  beforeDestroy
  destroyed
  activated
  deactivated
  
  vue3
  setup
  onBeforeMount
  onMounted
  onBeforeUpdate
  onUpdated
  onBeforeUnmount
  onUnmounted
  onActivated
  onDeactivated
  
  --------> onRenderTracked
  --------> onRenderTriggered
  ```
